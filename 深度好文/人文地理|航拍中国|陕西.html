# 航拍中国第三集——陕西

# 陕西

陕西位于中国西部,地跨南北,分陕南、关中、陕北三部分,秦岭以南是陕南,一派江南景象,秦岭以北是八百里秦川的关中平原,再往北是黄土高原,这里在抗日战争时期是多少青年人的梦想,陕北窑洞成了许多人对陕北的唯一影响,这里畜牧业和农业的分水岭.

# 山

# 秦岭

中国地理南北的分界线:秦岭淮河线.因其在关中以南,又名南山.因为秦岭中国有了南方、北方.秦岭最高峰太白山,是许多武侠小说不可缺少的角色.

# 终南山

秦岭北坡,就是赫赫有名对终南山.想必大家对这座山并不陌生,王维的《终南山》、韩愈的《南山诗》、孟郊的《游终南山》.

# 华山

华山,又称太华山,五岳之一,又称西岳.金庸武侠小说中出现的华山论剑的华山.中华民族的圣山,中华之华,源于华山.华山也是道教名山.是三十六洞中的第四洞.华山以险著称,有自古华山一条路之说.

# 水

# 黄河

中华名族的母亲河,黄河.河东就是山西,河西就是陕西.著名的壶口瀑布就位于陕西境内.

# 泾河、渭河

成语泾渭分明说的就是这两条河流,当夏天到来的时候,泾渭两河含沙量相同,颜色一样,当冬天到来时,渭河含沙量较少,出现明显的颜色差别,这就是泾渭分明的由来.黄河是中华名族的母亲河,渭河同是陕西的母亲河.

# 汉江

汉江亦称汉水.秦惠文王置汉中郡,后刘邦受项羽封于巴、蜀、汉中,又称汉王.统一天下后以汉为国号.汉朝前后400多年.原称华夏中原居民为汉人,汉人从此成为中国主体名族的通称.

# 动物

这里有彩色大熊猫、金丝猴、羚牛

# 大秦帝国

始皇帝统一六国,建立大秦帝国,死后留下秦兵马俑、秦始皇陵,至今尚未完全挖掘,始皇陵里日月星、辰江河湖海72w工人创造了今天的奇迹.始皇陵相当于63个故宫之大.

# 秦长城、明长城

# 乾陵

武则天、唐高宗

# 西安

西安即古长安.600年前改名西安,之前的1500年里被称作长安,是大唐帝国的国都.这里有大雁塔,是古丝绸之路的起点,比当时欧洲最大的城市君士坦丁堡的7倍,人口超过100多万,今天的日本京都就是仿长安而建.游西安,必登西安城墙.
